Problem 1 - Double Quantum Dot & Aharonov-Bohm Effect

Interactive tight-binding model for a Double Quantum Dot (Landauer formalism)

Problem Assignment

The Double Quantum Dot (DQD) consists of two consecutive Double-Barrier Resonant Tunneling Diodes (DBRTD). The system is schematically represented by two localized orbitals A and B (for VU = 0).

  1. Derive analytical expressions for the Landauer conductance using the Fisher-Lee relation.
  2. Plot the transmission function for several parameter choices and discuss the physical behavior.
  3. Add a third orbital C providing an additional tunneling channel.
  4. Introduce a magnetic field producing a flux represented by a phase 0 ≤ φ ≤ 2π (singular gauge).
  5. Discuss interference effects (Aharonov-Bohm effect).
  6. Identify antiresonances and compare them with the antiresonance phenomenon in two coupled pendulums.

Reference: D'Amato, Pastawski, Weisz (PRB 1989)
