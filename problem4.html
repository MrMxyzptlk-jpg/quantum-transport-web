Problem 4 — Polaronic State and Electron Transfer

Interactive model for resonant tunneling with decoherence (DP Model)

Problem Assignment

a) Polaronic State

Consider the two-state donor-acceptor model discussed in class, where the acceptor is coupled to a phonon mode of frequency ω₀ with energy \( \hbar \omega_0 (n + \frac{1}{2}) \).

Use the displaced harmonic oscillator solution to discuss:

  • The polaronic solution
  • Energy shifts
  • Physical interpretation (Franck-Condon effect)

Reference: Marder, Sec. 22.4.5

b) Marcus-Hush Electron Transfer

Discuss the kinetic rate constant:

\[ k = \frac{1}{\tau} \]

using Fermi's Golden Rule for donor-to-acceptor transfer. Compare with the Landau-Zener tunneling approach used by Marcus.

Reference: Cattena, Bustos-Marún, Pastawski (2010)

c) DBRTD with Local Phonon Coupling

  1. Extend the model to describe a DBRTD where the resonant state is coupled to a local phonon.
  2. Write the transmission equations for the case where the initial phonon occupation number \(n \neq 1\).
  3. Analyze the cases \( E \lessgtr \hbar\omega_0 \) and truncate the number of outgoing channels.
  4. Compare similarities and differences between the DBRTD with a local phonon and the DP decoherence model. Where do irreversible processes occur in each model?
Theoretical Background

Polaron shift

\[ \Lambda = \frac{\lambda^2}{\hbar\omega_0} \] \[ S = \left(\frac{\lambda}{\hbar\omega_0}\right)^2 \] Franck–Condon factors: \[ F_n = e^{-S}\frac{S^n}{n!} \] Transmission with phonons: \[ T(E) = \Gamma_L\Gamma_R \left|\sum_{n=0}^{N} \frac{F_n}{E-\varepsilon_0+\Lambda-n\hbar\omega_0+i\Gamma} \right|^2 \]
